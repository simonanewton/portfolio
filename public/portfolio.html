QuizApp

    Repository Website

QuizApp is a React-based web application that allows users to take accessible quizzes about several different subjects at numerous different difficulty levels. This was the first group project I participated in where we were asked to use React, so it was a challenge coordinating the implementation of different components, pages, and user authentication between different team members. I lead the creation of the main page user interface as well as the primary quiz component.

RetroFeed

    Repository Website

RetroFeed is a retro-styled social media web application where users are prompted to create an account, make individualized posts, and connect with other users. Our team took great inspiration from other social media platforms like Twitter and Facebook in the creation of user interface and user experience. I was in charge of creating and synchronizing the home feed interface where users could see a global listing of posts, as well as navigate to other helpful parts of the site.

Band Explorer

    Repository Website

Band Explorer is a music centered web application where users can search for musical performers to learn more about their most notable information and find out if they have any upcoming performances in the near future. I led a team of five in creating this application; this was the first project that we collaborated on at my web development bootcamp so we had to learn how to work with each other using the Git framework. I helped lead our team in the creation and overall design of this application so that users could get a curated experience when visiting the site.

Weather Dashboard

    Repository Website

Weather Dashboard is an informational weather web application that allows users to search by city to find current important weather details in addition to an upcoming five-day forecast. This application was one of my earlier homework assignments during my web development bootcamp, however I am proud of this application because of the attention to detail in recreating the original template and the thorough integration of the Open Weather Map API in the layout. I had to greatly utilize my newly learned experience with JavaScript to create conditional visual and informational features so that the site could be user-friendly.
